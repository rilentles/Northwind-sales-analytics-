# Northwind-sales-analytics
Northwind Sales Analytics project using Power BI. An interactive dashboard analyzing sales performance, revenue, orders, product trends, and shipping efficiency. Includes KPIs, time-based analysis, and business insights from a relational dataset.

# Overview

* This project is a sales analysis and business intelligence dashboard built using the Northwind sample database. 
* The objective was to explore the dataset using SQL, understand its structure and relationships, and then build an interactive Power BI dashboard to extract meaningful business insights.
* The analysis focuses on sales performance, order trends, product performance, customer distribution, and shipping efficiency using key metrics and KPIs.

# Tools & Technologies

* SQL (data exploration and filtering)
* Power BI (data modeling and dashboard development)
* DAX (data modeling and meassures)

# Project Workflow

## SQL Data Exploration

* Explored the Northwind database using SQL queries
* Analyzed relationships between orders, customers, products, and suppliers
* Identified key business metrics such as revenue, order volume, and product performance
* Prepared and validated data for visualization

## Power BI Dashboard Development

* Imported and modeled the data in Power BI
* Created relationships between key tables
* Built DAX measures for KPIs and aggregations
* Designed an interactive dashboard for business insights
  
## Dashboard Features

* Total revenue and order tracking
* Average monthly orders KPI
* Revenue breakdown by product and category
* Order distribution by country
* Shipping performance analysis
* Time-based sales trends

# Key Insights 

* A small number of products generate a large portion of total revenue
* Order volume varies significantly across months with noticeable spikes
* Sales are concentrated in a few key countries
* Shipping performance differs across carriers

# Purpose

This project demonstrates skills in SQL data analysis, data modeling, and business intelligence dashboard development using Power BI.
